# McacheClean

Extensão limpa as pastas var e generated na raiz de um projeto magento

Para rodar a extensão siga os passos abaixo:

CTRL+SHIFT+P

Digite o comando abaixo:

limparcache

## Em desenvolvimento a mensagem que aparece é apenas subliminar
